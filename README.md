# ai-movie
```
Привіт! Я AI артист та хочу створити короткометражний фільм. Допоможіть мені підібрати ідеї. Тому дій, як креативний копірайтер.
Тобі потрібно надати мені 5 ідей короткометражного фільму хронометражем до 5-х хвилин.
Ідеї подавай у форматі опису синопсису на 400 слів.
Ідеї фільму мають засновуватись на 2-х складових:
- Тема, що відбувається у фільмі. Саме вона задає весь сюжет.
- Герой, або герої фільму. Вони мають знаходитись в центрі сюжету та просувати його в перед.
```
